Colonization Clone

(v0.2.0 preview)

This game is in early version and is not "playable" yet. For keybindings see README.

This game uses some graphics from the original Colonization game. To use it you must own "Sid Meier's Colonization" game (from 1994). You can buy one at gog.com.

This version was tested on Firefox 40.0.2
